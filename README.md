## ChatGPT对话

记忆：默认记忆最后一次10分钟内消息，循环刷新记忆时间

7条消息记忆。发送【清除】可以重置所有记忆

更多环境变量，请查看.env.example

### 有两种方式

1、直接发送问题。。（默认模型gpt-4o-mini）  比如： 你会干什么？

2、 model+空格+问题。。（自己指定模型） 比如： gpt-4 你会干什么？
